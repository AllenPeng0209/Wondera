# 语言恋爱 + AI 恋爱游戏产品文档（V0.1）

## 1. 产品定位
- **一句话**：用恋爱剧情和 AI 角色陪练，让用户在沉浸式互动中高效习得外语。
- **双核心**：语言熟练度 + 好感度双成长；语言表现影响剧情分支，好感提升解锁更高阶语言任务。
- **差异化**：把 SRS 词汇记忆、分支对话剧场、语音纠错、写作批改融入恋爱进程，减少“学习感”，强化“代入感”。

## 2. 用户与场景
- **轻量学习者**：想提升口语/词汇但缺少动力；希望轻松互动而非标准课堂。
- **剧情/陪伴向用户**：喜欢恋爱互动、角色养成，接受语言学习作为玩法之一。
- **场景**：碎片时间（通勤/睡前）做词汇卡、对话闯关；周末集中完成语音/写作任务；节日限定剧情体验文化。

## 3. 目标与 KPI（首 3 个月）
| 指标 | 目标 | 备注 |
| --- | --- | --- |
| D7 留存 | ≥ 35% | 剧情分支 + 每日任务驱动回访 |
| 日均对话轮数 | ≥ 45 | 对话剧场 + 自由聊双入口 |
| 每日口语跟读完成率 | ≥ 60%（新手 7 天） | 语音陪练、即时纠错 |
| 付费转化 | ≥ 7% | 会员/道具组合 |

## 4. 核心体验框架
- **恋爱剧情**：多角色、多结局分支，对话选项使用目标语；好感与表达质量共振。
- **语言练习**：词汇 SRS + 语境卡；对话、语音、写作、文化任务闭环；即时纠错与地道表达示范。
- **动态难度**：根据词汇掌握、语音评分、写作质量自动调整词难度、语速、纠错严苛度。

## 5. 功能模块
### 5.1 词汇与记忆
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| 语境词卡 | 从剧情自动抽取关键词，展示发音、例句、近反义词，关联场景截图 | P0 |
| SRS 闪卡闯关 | 间隔复习；闯关奖励道具/好感度 | P0 |
| 发音/拼读训练 | 跟读打分，音素级反馈；弱项词单独复习 | P1 |

### 5.2 对话剧场
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| 分支对话 | 目标语选项影响剧情和好感；错误表达触发“温柔补救”或“直接点拨”两种纠错风格 | P0 |
| 即时纠错与示范 | 标注自然表达 vs 生硬表达，给替换句；可一键应用 | P0 |
| 语速/字幕模式 | 护航模式（慢速+实时字幕+词典热键）/沉浸模式（延迟字幕） | P1 |

### 5.3 AI 恋爱陪伴
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| 自由聊天 | 角色根据学习目标引导话题；轻量打卡 | P0 |
| 语音通话 | 模拟电话/语音备忘，实时纠错与鼓励 | P1 |
| 情感档案 | 记录用户情绪与偏好，调整话题和语气 | P1 |

### 5.4 写作与创作
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| 心情日记 | 目标语短文输入，AI 改写、评分、突出错误热点 | P1 |
| 共创文案 | 与 AI 合写信件/诗/约会计划，提供多档地道表达 | P2 |

### 5.5 文化沉浸与场景
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| 约会地点词包 | 咖啡馆/夜市/美术馆等场景词汇 + 任务 | P1 |
| 节日/热点剧情 | 实时文化事件限定分支与词汇挑战 | P2 |

### 5.6 成长与任务
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| 每日/每周任务 | 词汇量、口语时长、写作次数；完成提升好感并解锁剧情 | P0 |
| 角色特技 | 好感解锁“亲密纠错”（直接）或“温柔陪练”（间接）等风格 | P1 |
| 语言等级阶梯 | 对齐 CEFR；等级提升解锁高级语法/剧情段 | P1 |

### 5.7 社交与安全
| 功能 | 描述 | 优先级 |
| --- | --- | --- |
| CP 展示与排行榜 | 亲密度、学习时长、口语积分排名 | P2 |
| 协作对话赛 | 双人练习，AI 裁判评分 | P2 |
| 内容审核与边界 | 分级、敏感话题过滤、心理安全提示 | P0 |

## 6. 学习路径设计
- **新手 7 日护航**：基础词卡 + 慢速剧情 + 高频纠错；日均 10 分钟完成感。
- **进阶 30 日沉浸**：场景任务占比提升，自由对话比例增加，纠错更克制。
- **长期循环**：SRS 复习、每周写作、月度口语挑战、文化专题周。

## 7. 付费与激励
- **会员**：解锁高级剧情、多角色槽、语音评测高精度、离线词包下载。
- **道具**：好感加速卡、场景解锁券、稀有剧情钥匙、纠错风格切换券。
- **活动**：节日季卡、限时词汇包、排行榜赛季奖励。

## 8. 数据与反馈
- 仪表盘：词汇掌握度、口语流利度、错误热点、好感进度、剧情完成度。
- 反馈循环：每段对话后给一键吸收词汇/短评；每周总结 + 下周计划。
- 动态难度：基于完成率/评分调整词难度、语速、提示强度；必要时切回护航模式。

## 9. MVP → 迭代
- **MVP**：词卡 + SRS、分支对话剧场、即时纠错、每日任务、好感度系统。
- **迭代 1**：语音通话 + 跟读评分、日记批改、场景词包。
- **迭代 2**：文化限定剧情、多角色人格、协作对话赛。
- **迭代 3**：动态难度引擎、情感档案深度调节、赛事化活动。

## 10. 技术要点与风险
- **人格与语气模板**：Prompt 模块化，支持教学/恋爱/鼓励切换；好感驱动语气变化。
- **语音与纠错**：实时 ASR + 发音打分；多档纠错风格；异步写作批改。
- **本地化与合规**：内容分级、敏感词过滤、隐私保护；未成年人策略。
- **风险**：剧情与学习难度平衡；语音打分体验；文化内容审核。
