# 账号与云同步工作包

## 目标
把当前本地登录/本地存储升级为可跨设备的安全账号体系。

## 范围
- 账号注册/登录与找回
- 访客迁移与多设备同步
- 用户资料与偏好云端化
- 数据导出与注销

## 待完善清单
- [ ] 邮箱注册 + 验证 + 找回密码
- [ ] 密码安全（哈希、强度校验、登录限流）
- [ ] 访客账号升级与本地数据迁移
- [ ] 多设备同步与冲突策略
- [ ] 账号注销、数据导出/删除

## 数据同步重点
- 用户设置（昵称、性别、偏好、API 模式等）
- 聊天记录与角色进度
- 词库与复习进度

## 验收标准
- 同一账号在多设备可一致看到聊天/词库
- 访客可无损升级到正式账号
- 满足隐私合规（删除/导出）
