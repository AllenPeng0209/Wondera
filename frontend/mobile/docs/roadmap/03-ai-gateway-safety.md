# AI 网关与内容安全工作包

## 目标
将所有模型调用集中到服务端，保证成本可控与内容安全可管。

## 范围
- LLM / TTS 统一网关
- 计费与限额控制
- 内容安全与审核
- Prompt 管理与版本化

## 待完善清单
- [ ] 服务端代理 LLM/TTS（客户端不再携带密钥）
- [ ] 计费模型（按次/按 Token/按时长）与限额策略
- [ ] 输入/输出内容安全与敏感话题拦截
- [ ] 举报链路与风控开关
- [ ] Prompt 模板版本化与热更新

## 关键指标
- 单次会话成本、日均调用量、异常率
- 被拦截内容比例、用户举报量

## 验收标准
- 任何客户端无法直接调用第三方模型
- 安全策略可配置并可快速热更新
- 成本与配额可在后台控制
